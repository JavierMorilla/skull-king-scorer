# Product

## Register

product

## Users
Players of the Skull King card game in face-to-face game settings. Users need a fast, reliable utility to count bids, captures, bonuses, and rounds without pausing the game flow or getting distracted by heavy decorative elements.

## Product Purpose
Provide an effortless, real-time score keeping app for Skull King. It enables quick score calculations for online multiplayer rooms and local offline pass-and-play matches, making sure that players focus on playing the actual game rather than tracking score sheets.

## Brand Personality
* **Three words:** Minimalist, Tactile, Pure
* **Voice & Tone:** Crisp, utilitarian, native, distraction-free
* **Emotional Goals:** Evoke trust, absolute focus, and precision. The interface should feel like a native iOS system tool in dark mode.

## Anti-references
* Avoid overly decorated/cluttered pirate illustrations, parchment paper textures, or hand-drawn SVGs.
* Avoid typical light-saturated SaaS layouts, warm cream backgrounds, or generic web components.
* Avoid slow, complex page transitions or heavy decoration that gets in the way of fast data input.

## Design Principles
* **Utility Over Decoration:** Visual features must directly serve functional efficiency. Animation, color, and spacing should help the user input data faster and make fewer mistakes.
* **Pure Dark Mode Focus:** Utilize deep dark grays and blacks with high-contrast, clean system iconography. It should look premium and native to high-end mobile operating systems.
* **Instant Tactile Responses:** Every tap and adjustment must have immediate feedback (subtle active scaling, instant updates, and clean micro-interactions).

## Accessibility & Inclusion
* Standard WCAG AA contrast ratio compliance (>= 4.5:1 for body and interaction text) on all dark themes.
* Full support for system-reduced motion preferences to disable slide and scale animations while preserving simple color/opacity crossfades.
